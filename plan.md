## 最终目标

在 MATH-500 上给出有统计效力的结论：**自适应早停的 Power Sampling 重采样，能以明显更少的算力，达到（或接近）固定轮次重采样的准确率**，并量化"省算力 vs 抓救回"的权衡曲线，形成可写进报告/论文的核心表格与图。

## 已达成

- [x] 工程链路：p^α 目标的 MH 接受规则（`(α−1)Δ`，含 proposal 抵消）、答案护栏、完整性护栏、变长玩具分布收敛验证
- [x] 判分可信：换用 Qwen2.5-Math-7B-Instruct + 官方 CoT chat prompt，每题恰一个 `\boxed{}`（v3）
- [x] 自适应停止判据修正：接受增益收敛与连续拒绝分开统计（patience / rejection_patience）
- [x] 难度分层抽题（`--levels 4,5`，dataset_index 可追溯），题集区分度达标（初始 50%）
- [x] 首个"错→对"救回：512 窗口下 fixed 55% > 初始 50%（v5，q14）
- [x] 安全底线：v3–v6 全程零"对→错"
- [x] 权衡两端点量化：rp=4 省 19.5% 但丢救回；rp=8 保救回但只省 4.6%（v6）
- [x] 乱码退化治理：探针证实种子依赖（复跑 9/10 正常），实现复跑判定 `--degeneration-retries`（不污染 proposal 分布）
- [x] 离线测试 33/33；服务器迁移到新实例（westc / 4080 SUPER）并重建环境

- [x] **100 题三臂 paired 实验**（fixed / adaptive-rp4 / adaptive-rp8，512 窗口 + 复跑判定）——2026-07-20 完成（约 7 小时）。核心结论：rp8 与 fixed 逐题正确性 0 分歧、省 6.4% 算力；rp4 更省（15.9%）但系统性漏救回；bootstrap 95% CI [-2.0pp, +6.0pp] 跨零，方向对但未达统计显著；退化死重率降到 1%（复跑判定生效）
